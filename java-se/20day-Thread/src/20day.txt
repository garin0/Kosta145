1) Thread 개념
 Process: 현재 실행 중인 프로그램
 Thread: 프로세스 내의 세부적 실행 단위 
 (참고: 사전적인 의미: 실.. 여러개의 실이 모여 옷감이 만들어 진다.)
 Multi-Threading: 다수의 스레드가 동시에 실행
 ex) 동영상 플레이어 실행(프로세스), 영상 스레드 + 음향 스레드 + 자막 스레드 등이 멀티스레딩 된다.
 
 2) Thread 생성
  (1) extends Thread
  (2) implements Runnable
  
 3)
  start():  스레드를 실행 가능 상태로 보낸다.
  			이후 jvm에 의해 스케줄링을 받으면 실행상태로 바뀐다는 것을 의미한다.
  run(): 스레드 실행
  		 jvm에  의해 호출되어 스레드의 실행이 된다
  Thread의 우선순위
  자바 스레드 우선순위는 1~10까지 있고 
  기본(별도로 명시하지 않으면)은 5, 가장 낮은 순위는 1, 가장 높은 순위는 10이다.
  스레드의 우선순위를 설정하면 스케줄링시에 스케줄 확률이 높아지거나 낮아질수 있다.
